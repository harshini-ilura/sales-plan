CSV Format Guide

Required Columns:

  • email or email_address - Required

Optional Columns:

  • name or full_name - Contact name
  • first_name - First name
  • last_name - Last name
  • phone or phone_number - Phone number
  • company or company_name - Company name
  • domain or website - Company website
  • city or location - City
  • country - Country
  • state or province - State/Province
  • industry or sector - Industry
  • title or position - Job title
Example CSV
email,first_name,last_name,company_name,phone,city,country
john@example.com,John,Doe,Example Corp,+1234567890,New York,USA
jane@company.com,Jane,Smith,Company Inc,+9876543210,London,UK
Tips
  • CSV files can use comma or semicolon delimiters
  • First row should contain column headers
  • Column names are case-insensitive
  • Empty rows are automatically skipped
  • Rows without email addresses are skipped
  • Duplicate emails are skipped by default
  • Check server logs if import fails
